--> Adding Books After Deletion (Admin) 

If you delete any book from the Admin panel and want to insert it again, you can do it easily using the Book Insert content provided.
1] Simply copy the content from the provided files or JSON data
2] Paste it into the Insert Book form in the Admin panel
3] Click Submit / Insert to add the book back to the database
This allows you to restore deleted books quickly without manually recreating all details.
